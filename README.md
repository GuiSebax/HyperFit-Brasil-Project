# Hyper Fit Project

Hyper Fit Project é uma aplicação frontend desenvolvida em React para uma loja de roupas com foco em vestuário esportivo e itens voltados para academia, como camisas, shorts e tops. O projeto foi pensado para apresentar produtos de forma clara e atrativa, priorizando a experiência do usuário e destacando conforto, funcionalidade e estilo como valores centrais da marca durante a prática de atividades físicas.

A aplicação explora conceitos de componentização, reutilização de componentes e organização de layout, buscando uma interface responsiva e intuitiva para navegação em diferentes dispositivos. O foco do desenvolvimento esteve na construção de uma base sólida de frontend, com código legível, estrutura escalável e separação clara de responsabilidades, permitindo fácil evolução do projeto para funcionalidades futuras, como integração com backend, carrinho de compras ou sistema de pedidos.
